# Nvim configure sheet

## How to add a plugin

put a \<plugin-name\>.lua into plugin/
> theme ,functional plugin ... all of them are plugin, auto loaded

## How to add a support language 

1. use mason to manage lsp 
> or self-manage
2. add \<lsp-name\>.lua to lsp
> how to write it? follow https://github.com/neovim/nvim-lspconfig/tree/master/lsp
3. open lua/lsp.lua to enable
